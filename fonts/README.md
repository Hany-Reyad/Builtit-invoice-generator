# Fonts

This directory contains the Poppins font files required by the invoice PDF generator.

## Required files

| File                    | Weight  | Source                                                                 |
|-------------------------|---------|------------------------------------------------------------------------|
| `Poppins-Regular.ttf`   | 400     | [Google Fonts](https://fonts.google.com/specimen/Poppins)             |
| `Poppins-Bold.ttf`      | 700     | [Google Fonts](https://fonts.google.com/specimen/Poppins)             |

## How to obtain

Download directly from Google Fonts:

```
https://fonts.gstatic.com/s/poppins/v21/pxiEyp8kv8JHgFVrJJfecg.woff2
```

Or use the `google-fonts-downloader` npm package, or simply download the `.ttf` files
from the [Google Fonts GitHub repository](https://github.com/google/fonts/tree/main/ofl/poppins)
and place them in this directory.
